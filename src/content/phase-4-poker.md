## L1: 確率論的基礎 — アウツと勝率計算

### 1.1 アウツ（Outs）の定義
アウツとは、残りのデッキの中で自分の手を改善できるカードの枚数。

**例1: フラッシュドロー**
- 手札: A♠ K♠
- フロップ: Q♠ 7♠ 2♦
- 現在: ハイカードAKのみ（フラッシュドロー中）
- アウツ: 残りの♠ = 13 - 4(見えてる♠) = **9枚**

### 1.2 勝率の近似計算（2倍・4倍の法則）

**ターンでの勝率**: アウツ × 2 = 近似勝率(%)
**フロップでの勝率**: アウツ × 4 = 近似勝率(%)

| アウツ | フロップ→リバー(実測) | 4倍近似 | 誤差 |
|-------|---------------------|--------|------|
| 4 (ガットショット) | 16.5% | 16% | -0.5% |
| 8 (オープンエンド) | 31.5% | 32% | +0.5% |
| 9 (フラッシュドロー) | 35.0% | 36% | +1.0% |
| 12 (FD+ガット) | 45.0% | 48% | +3.0% |
| 15 (FD+OE) | 54.1% | 60% | +5.9% |

**限界**: アウツが13以上になると誤差が大きくなる。正確な計算には組合せ論が必要。

### 1.3 正確な確率計算（組合せ論）

**例: フラッシュドロー完成確率（ターン→リバー）**

```
全未知カード: 52 - 2(手札) - 4(コミュニティ) = 46枚
アウツ: 9枚
ヒットしない確率: (46-9)/46 × (45-9)/45 = 37/46 × 36/45 = 0.643
ヒットする確率: 1 - 0.643 = 0.357 = 35.7%
```

## L2: ポットオッズと期待値

### 2.1 ポットオッズの定義
ポットオッズ = コール額 / (ポット額 + コール額)

**例**: ポットに$100、相手が$50にベット。コールに必要な金額は$50。
```
ポットオッズ = 50 / (100+50) = 50/150 = 1/3 = 33.3%
すなわち、勝率が33.3%以上あればコールは長期的にプラス期待値。
```

### 2.2 コール判断の期待値計算

**EV(コール) = (勝率 × 獲得額) - ((1-勝率) × 損失額)**

**例（上記の状況 + フラッシュドロー）**:
```
勝率 = 35.7%（L1計算より）
EV = (0.357 × $150) - (0.643 × $50)
   = $53.55 - $32.15
   = $21.40 ← 長期的に1回のコールあたり$21.40のプラス
結論: コールすべき
```

### 2.3 インプライドオッズ（期待される将来のベットも含める）

単純なポットオッズが不十分でも、ハンドが完成した後に相手からさらにチップを引き出せる場合はコールが正当化される。

**修正EV**: EV = (勝率 × (ポット + 将来獲得額)) - ((1-勝率) × 損失額)

**例**: ポット$30、相手が$20にベット。コール額$20。
4倍の法則で勝率16%（ガットショット）。
ポットオッズ: 20/(30+20) = 40%。16% < 40% → 単純計算ではフォールド。
しかし、ヒットしたら追加$80は引き出せると読む:
```
EV = (0.16 × ($50 + $80)) - (0.84 × $20)
   = (0.16 × $130) - $16.80
   = $20.80 - $16.80
   = $4.00 ← インプライドオッズ込みではプラス
```

## L4: ディーラーの実務計算

### 4.1 ポット計算のステップバイステップ

**例**: プリフロップ、SB=$1, BB=$2。UTGが$6にレイズ、COがコール、BBがコール。

```
Step 1: プリフロップ強制ベット
  SB $1 + BB $2 = $3
Step 2: UTGレイズ $6
  ポット = $3 + $6 = $9
Step 3: COコール $6
  ポット = $9 + $6 = $15
Step 4: BBコール $4（すでに$2入っているので追加$4）
  ポット = $15 + $4 = $19
Step 5: レーキ（5%、上限$3）
  レーキ = min($19 × 5%, $3) = $0.95
  有効ポット = $19 - $0.95 = $18.05
```

### 4.2 サイドポットの計算

**例**: Aが$100オールイン、Bが$200持っている、Cが$300持っている。Bがレイズ$200、Cがコール。

```
メインポット: 3人 × $100 = $300
  対象: A, B, Cの3人
サイドポット1: Bの追加$100 + Cの追加$100 = $200
  対象: B, Cの2人
総ポット: $300 + $200 = $500
レーキ: 5% = $25
有効総ポット: $475
```

### 4.3 レーキ計算のバリエーション

| レーキ方式 | 計算方法 | 例 |
|-----------|---------|-----|
| タイム（時間制） | 30分ごとに固定額 | $6/30min |
| ポット（割合制） | ポットの5%（上限$3） | $100ポット→$3 |
| ジャックポットドロップ | ポットから$1-2を別途徴収 | バッドビート基金 |

## L5: トーナメント数学

### 5.1 ブラインド構造の指数関数的成長

**例**: 標準的なWSOP式構造
```
Level 1: 100/200
Level 2: 100/200 (25 ante)
Level 3: 150/300 (25 ante)
Level 4: 150/300 (50 ante)
Level 5: 200/400 (50 ante)
...
Level 10: 500/1,000 (100 ante)
```

ブラインドの成長率 = (次のBB) / (現在のBB) = 一般的に**1.1-1.25倍/レベル**

### 5.2 ICM（Independent Chip Model）入門

チップの価値は線形ではなく、**チップが多いほど1枚あたりの価値が下がる**。

**例**: 3人トーナメント。賞金: 1位$100、2位$50、3位$0。
```
A: 5,000 chips
B: 3,000 chips  
C: 2,000 chips

Aの期待値 = (5000/10000) × $100 + (確率に応じて) ...
ICM計算の結果、Aのチップの期待値は約$63（単純比例なら$50）
```

**実務的ポイント**: バブル（入賞直前）では、ショートスタックはタイトに、ビッグスタックはアグレッシブに。

## L6: ディーリング実務 — エラー防止と正確性

### 6.1 一般的なミスと防止策

| 状況 | よくあるミス | 防止策 |
|------|-----------|--------|
| フロップ配布 | 4枚配ってしまう | バーンした後、3枚を一度に配らない（1枚ずつ3回） |
| ターン配布 | フロップと合わせて配る | バーン後、明確に1枚だけを配る |
| ポット計算 | サイドポットの計算ミス | オールイン状況では「プレイヤー別のチップ総額」を確認 |
| レーキ計算 | パーセンテージ誤差 | 小額のポットでは頭で計算、大額では電卓 |

### 6.2 正確なポットサイズ把握のテクニック

1. ベットが発生したら即座にポット合計を更新する
2. 複数のベットがある場合、積み重なったチップの山の色別価値を数える
3. 混乱したら「ポット $XX」と宣言して確認を取る
4. 全員がアクションを終えるまでは最終確定しない

## 修了試験準備問題

### 計算問題（全問、計算過程を記述すること）

**Q1**: ポット$80、相手が$40にベット。あなたの勝率は25%。
    コールのEVは？

**Q2**: フロップでオープンエンドストレートドロー（アウツ8枚）。
    リバーまでに完成する正確な確率は？（組合せ論で計算）

**Q3**: プリフロップ、SB$1, BB$2。UTGレイズ$8、MPコール$8、あなたはBBでKK。
    レイズ額は$35。ポットはいくら？レーキ5%（上限$4）はいくら？

**Q4**: 4人トーナメント、賞金1位$200、2位$100、3位$50、4位$0。
    チップ: A=6,000 / B=4,000 / C=3,000 / D=2,000。
    ICMの考え方に基づくと、Aのチップ1枚の期待値は？（概算で良い）

### 解答

**A1**: EV = (0.25 × $120) - (0.75 × $40) = $30 - $30 = $0
    ブレークイーブン。僅かでもインプライドオッズがあればコール。

**A2**: 全未知カード = 52-2-4 = 46枚。アウツ8枚。
    完成確率 = 1 - (38/46 × 37/45)
           = 1 - (1,406/2,070)
           = 1 - 0.679
           = 0.321 = **32.1%**
    （2倍の法則だと32%で近似は優れている）

**A3**: ポット = $1 + $2 + $8 + $8 + $35 = $54（あなたの$35含む）
    あなたの実質コール額 = $35 - $2(BB) = $33
    レーキ = min($54 × 5%, $4) = min($2.70, $4) = **$2.70**
    有効ポット = $54 - $2.70 = **$51.30**

**A4**: Aのチップ = 6,000/15,000 = 40%
    単純比例なら $350 × 40% = $140
    ICMでは1位確率が40%未満、2位確率も考慮 → 約$120-$130が現実的。
    （厳密なICM計算には専用ソフトが必要）
