# ディーラー物理的技法 完全ガイド

ディーラーに必要な身体的操作スキルを体系的に解説。

## 1. カード操作（Card Handling）

### 1.1 ウォッシュ（Wash / Scramble）
**目的**: 新カードまたは一度のゲームで使用したカードを初期混合する。

**手順**:
1. 全カードをテーブル上に裏向きに広げる（約60cm四方）
2. 両手のひらを使い、カードを円を描くようにかき混ぜる
3. 時計回りと反時計回りを交互に行う（約15-20秒）
4. カードが均等に混ざったら中央に集める

**注意点**:
- カードがテーブルから落ちないように
- 手のひら全体を使い、指先だけでは傷をつける可能性
- カードが折れたり曲がったりしないように

**使用ゲーム**: ポーカールーム（新デッキ導入時）、BJ（デッキ交換時）

### 1.2 リフルシャッフル（Riffle Shuffle）
**目的**: カードを高速・高精度にシャッフルする基本技法。

**手順**:
1. デッキを2つの等しい山に分割
2. 各山を親指と中指で持ち、人差し指を上部に添える
3. 両方の山の角をテーブルに付け、親指でカードの角をリリース
4. 左右のカードが1枚ずつ交互に落ちるようにする
5. カードが完全に交差したら、両側から押し込んで1つの山にする
6. アーチ（ブリッジ）: 山を曲げてカードをアーチ状にリリースし、カード同士を噛み合わせる

**上達のコツ**:
- まずは10枚ずつの小さな山から練習
- カードの角を持つ位置は一定に
- リリースのタイミングを均等に
- 練習の目安: 30秒で完璧なリフル×2回

**使用ゲーム**: 全ゲーム

### 1.3 ストリップシャッフル（Strip / Ripple Shuffle）
**目的**: リフルシャッフルの前にカードの塊を切り離す。

**手順**:
1. デッキを片手に持つ
2. もう一方の手で上部から5-10枚程度の塊を引き抜く
3. 引き抜いた塊を手前または別の位置に置く
4. 残りの山から同様に塊を引き抜く
5. すべての塊をランダムな順序で再積み重ねる

### 1.4 オーバーハンドシャッフル（Overhand Shuffle）
**目的**: 最も基本的なシャッフル。初心者向け。

**手順**:
1. デッキを片手に持つ
2. もう一方の手で上部から塊を取り、元の山の上に重ねる
3. これを繰り返す
4. 塊のサイズを変えることでランダム性を確保

### 1.5 ストリップカット（Strip Cut）
**目的**: シャッフル後にカットを行い、不正防止。

**手順**:
1. デッキの上部約1/3をカット
2. それを横に置く
3. 残りの2/3の上に最初の山を載せる
4. またはプレイヤーにカットを促す（カットカードを使用）

## 2. ディーリング技法（Dealing Techniques）

### 2.1 ピッチディーリング（Pitch Dealing）
**目的**: カードを1枚ずつプレイヤーに投げるように配る。主にシングル/ダブルデッキBJで使用。

**正しいフォーム**:
1. **グリップ**: デッキを利き手で持ち、親指は上部、中指と薬指は側面
2. **ステンス**: 足を肩幅に開き、テーブルに対してやや斜め
3. **モーション**:
   - 手首のスナップを使い、カードに回転（スピン）をかける
   - カードは胸の高さからリリース
   - カードは水平に保ち、テーブルに平行に飛ばす
4. **着地**: カードはプレイヤーの前のテーブルに水平に着地

**一般的なミス**:
- 手首ではなく腕全体を使う → カードがふらつく
- カードに回転がかかりすぎる → テーブルから落ちる
- カードが縦に立って飛ぶ → プレイヤーが見づらい
- デッキの持ち方が不安定 → 複数枚一度に出る

**練習方法**:
1. 最初は1枚のカードを自分の前に投げる練習（10分）
2. 次にデッキから1枚ずつ抜き出して投げる（20分）
3. 2枚のカードを連続で配る（5-10分）
4. 徐々に距離と速度を上げる

### 2.2 シューディーリング（Shoe Dealing）
**目的**: マルチデッキBJおよびバカラで使用。

**手順**:
1. シューにカードを正しい向きでセット
2. 左手でシューの側面を固定
3. 右手の親指で一番上のカードを前方に押し出す
4. カードが完全に出たら人差し指と中指で摘む
5. プレイヤーの前にカードを置く

**コツ**:
- 一度に1枚だけ出す（2枚同時はミス）
- カードを引っ掛けないようにシューの角度を調整
- 一定のリズムで配る（早すぎず遅すぎず）

**使用ゲーム**: マルチデッキBJ（4-8デッキ）、バカラ

### 2.3 バカラディーリング
**特徴**: バカラではプレイヤーとバンカーに2枚ずつ配る。

**手順**:
1. 最初のカード: プレイヤー（表向き）
2. 2枚目のカード: バンカー（表向き）
3. 3枚目のカード: プレイヤー（表向き）
4. 4枚目のカード: バンカー（表向き）
5. 必要に応じて3枚目の追加カードを配布

**パレット使用**: 高額バカラでは木製のパレット（ヘラ）を使用してカードを配る

## 3. チップ操作（Chip Handling）

### 3.1 チップの切り方（Cutting）
基本中の基本。チップの山を2つに分割する。

**手順**:
1. チップの山を両手で包み込むように持つ
2. 親指で山の中央を押し、2つの山に分ける
3. 両方の山をテーブルに置く
4. 一方の山をもう一方の上に重ねる

### 3.2 チップシャッフル（Chip Shuffling）
**目的**: 2つのチップの山を1つに混ぜる。

**手順**:
1. 左手と右手にそれぞれチップの山（10-20枚）を持つ
2. 両方の山の端をテーブルにつける
3. 親指でチップを1-3枚ずつリリースし、交互に落とす
4. すべてのチップが落ちたら、両側から押し込み整える

### 3.3 チップスタッキング（Stacking）
**目的**: チップを美しく整列させる。

**基本ルール**:
- 1スタック = 20枚
- 高額チップ（$100以上）は別スタックに
- テーブルに並べる際は色別にグループ化

**テクニック**:
- スタックを机にトントンと打ち付けて揃える
- スタックの両側から人差し指で押さえて整える
- スタックを持ち上げて、机の上でカツンと揃える

### 3.4 チップカット（Cutting Chips / Splitting Stacks）
**目的**: 1スタック（20枚）を素早く正確に2分割する。

**基本の10-10カット**:
1. スタックを左手で持つ
2. 右手の親指で真ん中あたりを押す感覚で10枚を分割
3. 分割した10枚を右手で取り出す
4. 2つの10枚スタックをテーブルに置く

**コツ**:
- 3枚の指（人差し指・中指・薬指）で側面をガイド
- 親指で真ん中を押し出す感覚
- テーブル上で行うとカウントしやすい

### 3.5 カラーチェンジ（Color Change）
**目的**: 複数の低額チップを高額チップに交換する。

**手順**:
1. 低額チップを計算する（ラックから取り出し、カウント）
2. 合計額に対応する高額チップをラックから取り出す
3. 低額チップをラックにしまい、高額チップをプレイヤーに渡す
4. すべての動作をクリアに見せる（カメラ対策）

## 4. ペイアウト技法（Payout Mechanics）

### 4.1 ブラックジャックペイアウト
**3:2計算のコツ**:
- ベット額の半分を計算 → それにベット額を加算
- 例: $25 → $12.50 + $25 = $37.50
- 例: $15 → $7.50 + $15 = $22.50

**チップの組み合わせ**:
- まず高額チップから置く
- 次に低額チップを上に重ねる
- 賞金と元のベットは別々に扱う

### 4.2 クラップスペイアウト
- プレイス6/8: $6単位で7:6 → $6→$7、$12→$14
- プレイス5/9: $5単位で7:5 → $5→$7、$10→$14
- プレイス4/10: $5単位で9:5 → $5→$9、$10→$18

### 4.3 ルーレットペイアウト
- チップをレイアウト上の正しい番号に置く
- 複数の勝ちベットがある場合、まずすべてのチップを回収してから支払う
- 36枚チップの色分け管理

## 5. エラー防止テクニック

### ダブルチェックの習慣
1. カードを配る前に: ベットが正しいか確認
2. ペイアウト前に: 勝ち手とベット額を再確認
3. チップを渡す前に: 額面を声に出して確認

### 一般的なエラーと対策
| エラー | 原因 | 対策 |
|--------|------|------|
| チップの過払い | ベット額の誤認 | 声に出して確認 |
| カードの2枚同時 | グリップ不良 | 親指の位置を固定 |
| 配る順序ミス | 左→右の徹底不足 | 開始前に指で方向を指す |
| バーンの忘れ | ルーティンの不備 | 手順カードを設置 |
