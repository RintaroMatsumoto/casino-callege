## 1. 確率論の基礎

### 1.1 試行・事象・確率の公理
- **試行 (Trial)**: サイコロを1回振る、ルーレットを1回回す
- **事象 (Event)**: 試行の結果（例: サイコロの目が3）
- **確率 P(A)**: 事象Aが起こる確率。0 ≤ P(A) ≤ 1

**コルモゴロフの公理**:
1. 任意の事象Aについて P(A) ≥ 0
2. 全事象Ωについて P(Ω) = 1
3. 排反事象の和: P(A∪B) = P(A) + P(B)（AとBが同時に起きない場合）

### 1.2 条件付き確率

P(A|B) = P(A∩B) / P(B)  … 「Bが起きた条件下でAが起きる確率」

**カジノでの例**: クラップスで「カムアウトロールが4以上だった」という条件下で「パスラインが勝つ」確率。

### 1.3 ベイズの定理

P(A|B) = P(B|A) × P(A) / P(B)

**追跡例**: あるカジノでカードカウンターが検出される確率。
- P(カウンター) = 0.05（全プレイヤーの5%がカードカウンター）
- P(検出|カウンター) = 0.70（70%の確率で検出される）
- P(検出|非カウンター) = 0.01（非カウンターが誤検出される確率1%）

P(カウンター|検出) = (0.70 × 0.05) / (0.70×0.05 + 0.01×0.95) = 0.035 / 0.0445 = 78.7%

## 2. 期待値 (Expected Value)

### 2.1 定義と計算

E[X] = Σ(xᵢ × P(xᵢ))

カジノでは「1ベットあたりの平均損益」を表す。

**例: アメリカンルーレット $1ストレートアップ**
```
E = $35 × (1/38) + (-$1) × (37/38)
  = $0.921 - $0.974
  = -$0.0526
```

### 2.2 期待値の線形性

E[aX + bY] = aE[X] + bE[Y]

**応用**: 複数のベットを同時に行った場合の期待値は、個々の期待値の和。

## 3. 分散と標準偏差

### 3.1 分散の定義

Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²

**標準偏差**: σ = √Var(X)

### 3.2 ルーレットでの計算例（$1ストレートアップ、米国）

```
E[X] = -$0.0526（上記より）
E[X²] = ($35)² × (1/38) + (-$1)² × (37/38)
      = $1,225 × 0.0263 + $1 × 0.9737
      = $32.24 + $0.97
      = $33.21

Var(X) = $33.21 - (-$0.0526)² = $33.21 - $0.0028 = $33.21
σ = √$33.21 = $5.76
```

### 3.3 BJの標準偏差（$1ベット、6デッキS17）

| 結果 | 確率 | ペイアウト |
|------|------|-----------|
| BJ | 4.75% | +$1.50 |
| 通常勝ち | 42.22% | +$1 |
| プッシュ | 8.48% | $0 |
| 通常負け | 44.55% | -$1 |

```
E[X] = (0.0475 × $1.50) + (0.4222 × $1) + (0.0848 × $0) + (0.4455 × -$1)
     = $0.071 + $0.422 + $0 - $0.446
     = $0.047（1ベットあたりの期待値、約4.7セント）

E[X²] = (0.0475 × $2.25) + (0.4222 × $1) + (0.4455 × $1)
      = $0.107 + $0.422 + $0.446
      = $0.975

Var(X) = $0.975 - ($0.047)² = $0.973
σ = √$0.973 = $0.987

したがって、平均-$0.047の期待値に対して、
標準偏差は約$0.99（1ベットあたり）。
```

### 3.4 全ゲームの標準偏差比較（$1ベットあたり）

| ゲーム | 期待値 | 標準偏差 | 変動係数(σ/|E|) |
|-------|--------|---------|----------------|
| BJ（基本戦略） | -$0.006 | $0.99 | 165 |
| ルーレット(米国) ストレートアップ | -$0.053 | $5.76 | 109 |
| ルーレット(欧州) ストレートアップ | -$0.027 | $5.84 | 216 |
| クラップス パスライン | -$0.014 | $1.00 | 71 |
| クラップス Any 7 | -$0.167 | $4.99 | 30 |
| バカラ Banker | -$0.011 | $0.96 | 89 |

**洞察**: BJは期待値が最も小さいが標準偏差は大きい → 短期的な変動が大きい。
一方、クラップスのAny 7は期待値が悪いが分散は大きい → 短期的に勝つこともあるが長期的に確実に負ける。

## 4. 大数の法則 (Law of Large Numbers)

### 4.1 弱い大数の法則

試行回数 n が増えるにつれ、標本平均 X̄n は母平均 μ に確率収束する。

**実務的意味**: 
- 100回のBJ: 結果はまだ運に大きく左右される
- 100,000回のBJ: 結果はハウスエッジに非常に近づく
- 1,000,000回のBJ: 結果はほぼ確実にハウスエッジ×ベット数

### 4.2 収束速度の可視化

| 試行回数(n) | BJ結果の95%信頼区間($1ベット) |
|-----------|---------------------------|
| 100 | -$0.60 ± $19.60 |
| 1,000 | -$6.00 ± $62.00 |
| 10,000 | -$60.00 ± $196.00 |
| 100,000 | -$600.00 ± $620.00 |
| 1,000,000 | -$6,000 ± $1,960 |

```
95%信頼区間 = n × E[X] ± 1.96 × σ × √n
n=100の場合: 100×(-$0.006) ± 1.96 × $0.99 × 10
           = -$0.60 ± $19.40
```

**重要な結論**: 10,000回プレイしても95%の確率で期待値の約3倍の誤差範囲。短期では運の要素が支配的。

## 5. 中心極限定理 (Central Limit Theorem)

### 5.1 定理の内容

どんな分布でも、サンプルサイズ n が大きくなるにつれて、標本平均の分布は正規分布 N(μ, σ²/n) に近づく。

### 5.2 カジノでの応用

ルーレット1回の分布は二項分布だが、100回の平均の分布は**ほぼ正規分布**。

**これにより**:
- 試行結果の確率的予測が可能
- 信頼区間の計算が可能
- リスクオブ・ルイン（破産確率）の計算が可能

## 6. 信頼区間 (Confidence Interval)

### 6.1 定義と計算

95%信頼区間: μ ± 1.96 × σ/√n

**例: 1,000回のBJ（$10ベット、HE=0.5%）**
```
μ = 1,000 × (-$10 × 0.005) = -$50
σ_per_hand = $10 × 0.99 = $9.90
σ_total = $9.90 × √1,000 = $313.05

95%信頼区間: -$50 ± 1.96 × $313.05
            = -$50 ± $613.58
            = [-$663.58, $563.58]
```

つまり、1,000ハンドのBJでは、95%の確率で結果が-$664から+$564の間に収まる。初心者が「勝てた！」と思っても、それは単なる統計的変動の範囲内である。

## 7. 統計的検定の基礎

### 7.1 帰無仮説と対立仮説

- H₀: ディーラーに偏りはない（確率通り）
- H₁: ディーラーに偏りがある

### 7.2 カイ二乗検定の実例

ルーレットホイールの偏りを検出する。

**データ**: 1,000回のスピンで「00」が30回出現（期待値: 1,000/38 = 26.3回）

```
期待度数: 26.3
観測度数: 30
カイ二乗値 = Σ(O-E)²/E = (30-26.3)²/26.3 = 0.52

自由度1、有意水準5%の臨界値 = 3.84
0.52 < 3.84 → 帰無仮説を棄却できない。
→ このデータだけでは偏りがあるとは言えない。
```

## 8. リスクオブ・ルイン (Risk of Ruin)

### 8.1 定義
所定のバンクロール、ベットサイズ、ゲーム条件で、無限にプレイした場合にバンクロールがゼロになる確率。

### 8.2 計算式（BJの場合）

R = ((1 - (W/L)) / (1 + (W/L)))^(B/σ²)

ただし:
- W = 勝率
- L = 負率
- B = バンクロール（単位: ベット）
- σ = 標準偏差

**簡易近似（BJの場合）**:
R ≈ exp(-2 × EV × B / σ²)

**例**: バンクロール$5,000、$10ベット、HE=0.5%、σ=$0.99
```
B = 500単位
EV = 0.005（ハウスエッジ）
σ per hand = $0.99
R ≈ exp(-2 × 0.005 × 500 / 0.99²)
  ≈ exp(-5.10)
  ≈ 0.006 = 0.6%
```

**実用的なルール・オブ・サム**:
| ゲーム | 必要バンクロール（ベットサイズの倍数） |
|-------|-------------------------------------|
| BJ（基本戦略） | 200-500倍 |
| BJ（カードカウンティング） | 500-1,000倍 |
| クラップス（パスライン） | 100-200倍 |
| ルーレット（米国） | 500-1,000倍 |
| ポーカー（キャッシュゲーム） | 5,000-10,000倍（BB単位） |
