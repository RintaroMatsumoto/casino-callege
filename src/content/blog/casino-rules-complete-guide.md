## カジノルール完全ガイド

カジノには数多くのゲームがあるが、世界中のカジノでプレイされている主要ゲームは5つに絞られる。本ガイドでは、ブラックジャック、ルーレット、バカラ、クラップス、ポーカーのルールを完全解説する。

### ブラックジャックのルール

ブラックジャックは世界で最も人気のあるカジノゲーム。プレイヤーとディーラーが対戦し、21に近い方が勝ちというシンプルなルールながら、戦略性の高さから根強い人気を誇る。

**基本ルール**:
- カードの価値はA=1または11、絵札(J/Q/K)=10、数字はそのまま
- プレイヤーとディーラーに2枚ずつ配られる
- プレイヤーはカードを引き（ヒット）、止める（スタンド）を選択
- 21を超えるとバースト（即敗北）
- ディーラーは17以上でスタンド（強制）

**主なアクション**:
| アクション | 内容 |
|-----------|------|
| ヒット(Hit) | カードをもう1枚引く |
| スタンド(Stand) | 現在の手札で勝負 |
| ダブルダウン | 賭け金を2倍にし、1枚だけ引く |
| スプリット | 同じ数字のペアを2つの手に分ける |
| サレンダー | 最初の2枚で賭け金の半分を放棄して降りる |

**ハウスエッジ**: 基本戦略を使えば約0.5%（カジノゲーム中最も低い部類）

:::

### ルーレットのルール

ルーレットは、回転するホイールに球を投げ入れ、止まった番号を当てるゲーム。シンプルながら高い興奮を味わえる。

**基本ルール**:
- 37（欧州式）または38（米国式）のポケットがあるホイールを回す
- ディーラーが球を投げ入れ、止まった番号が勝ち
- プレイヤーはどの番号に止まるかを予想して賭ける

**ベットの種類**:
| ベット | 配当 | 説明 |
|-------|------|------|
| ストレートアップ | 35:1 | 単一の番号に賭ける |
| スプリット | 17:1 | 2つの番号の間に賭ける |
| ストリート | 11:1 | 横一列3つの番号に賭ける |
| コーナー | 8:1 | 4つの番号が交わる点に賭ける |
| 赤/黒 | 1:1 | 色に賭ける |
| 奇数/偶数 | 1:1 | 奇数か偶数に賭ける |
| ダズン | 2:1 | 1-12、13-24、25-36のいずれかに賭ける |

**ハウスエッジ**: 欧州式2.70%、米国式5.26%

:::warn
米国式ルーレットは00（ダブルゼロ）があるため、ハウスエッジが約2倍になる。できるだけ欧州式を選ぶこと。
:::

### バカラ（プントバンコ）のルール

バカラはマカオのカジノ収入の91%を占める、世界最大のカジノゲーム。ルールは非常にシンプルで、プレイヤーは「Playerが勝つか」「Bankerが勝つか」「引き分けか」の3択に賭けるだけ。

**基本ルール**:
- プレイヤーとバンカーに2枚ずつカードを配る
- 9に近い方が勝ち（10以上は10を引く：例 7+6=13→3）
- 状況に応じて3枚目を引く（タブローという固定ルールで自動決定）

**配当**:
| ベット | 配当 | ハウスエッジ |
|-------|------|------------|
| Player勝ち | 1:1 | 1.24% |
| Banker勝ち | 1:1（ただし5%コミッション） | 1.06% |
| Tie（引き分け） | 8:1または9:1 | 14.36% |

:::info
バカラのタブロー（3枚目ドロー規定）の覚え方は「バカラのタブローを完全暗記する方法」で詳しく解説している。
:::

### クラップスのルール

クラップスは2つのサイコロを使うゲームで、カジノで最も興奮するゲームと言われる。多くのベットオプションがあるが、基本は「パスライン」ベットから始めるのがよい。

**基本の流れ**:
1. カムアウトロール（最初のロール）:
   - 7か11 → パスラインの勝ち（ナチュラル）
   - 2、3、12 → パスラインの負け（クラップス）
   - 4、5、6、8、9、10 → その数字が「ポイント」に
2. ポイント成立後:
   - ポイントと同じ数字が出たら勝ち
   - 7が出たら負け（セブンアウト）

**おすすめベット**:
| ベット | ハウスエッジ |
|-------|------------|
| パスライン | 1.41% |
| オッズベット | 0%（最高） |
| プレイス6/8 | 0.46% |
| フィールド | 2.78% |

### ポーカー（テキサスホールデム）のルール

ポーカーは他のカジノゲームと異なり、プレイヤー同士が対戦するゲーム。カジノ（ハウス）はレーキ（手数料）を取るだけで、ゲームには参加しない。

**基本ルール（テキサスホールデム）**:
1. 各プレイヤーに2枚の手札（ホールカード）が配られる
2. 5枚のコミュニティカードがテーブルに公開される（フロップ3枚→ターン1枚→リバー1枚）
3. 手札2枚＋コミュニティ5枚のうち、最も強い5枚の組み合わせで勝負

**ハンドランキング**:
| 順位 | ハンド | 例 |
|-----|-------|-----|
| 1 | ロイヤルフラッシュ | A-K-Q-J-10（同スート） |
| 2 | ストレートフラッシュ | 5-6-7-8-9（同スート） |
| 3 | フォーカード | A-A-A-A-X |
| 4 | フルハウス | K-K-K-Q-Q |
| 5 | フラッシュ | 同スート5枚 |
| 6 | ストレート | 連続する数字5枚 |
| 7 | スリーカード | 7-7-7-X-X |
| 8 | ツーペア | A-A-Q-Q-X |
| 9 | ワンペア | 10-10-X-X-X |
