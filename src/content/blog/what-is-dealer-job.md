## カジノディーラーの仕事内容とは？

「カジノディーラーって具体的に何をするの？」「華やかに見えるけど実際は大変？」そんな疑問に答えるため、ディーラーの仕事内容を徹底解説する。

### ディーラーの1日

| 時間 | 業務 |
|------|------|
| 出勤30分前 | 更衣室で制服に着替え、身だしなみチェック |
| 出勤 | タイムカード、ピットボスに挨拶、テーブル担当の確認 |
| セッション1 | 45〜60分のゲーム進行（休憩なし） |
| 休憩 | 20分〜30分のブレイク。軽食や水分補給 |
| セッション2 | 再びテーブルに立ちゲーム進行 |
| 以下繰り返し | 8時間シフトで4〜5セッション |
| 退勤 | チップ精算、報告書提出、着替え |

### 主な業務内容

- **ゲーム進行**: カード配布、ルーレットの球投げ、ダイスの管理。ルールに従った進行が求められる。
- **チップ管理**: ベットの受け付け、ペイアウト計算と支払い、チップトレイの在庫管理。
- **不正防止**: カードのマーキング、チップの不正操作、コラボレーション（プレイヤー同士の共謀）の監視。
- **カスタマーサービス**: プレイヤーへの挨拶、ルール説明、クレーム対応。笑顔とプロフェッショナルな態度を保つ。
- **テーブルメンテナンス**: ゲーム間のカード交換、テーブルの清掃、備品の補充。

### 向いている人

- **几帳面で正確**: 金銭を扱うため、ミスが許されない。計算が正確で、確認を怠らない人。
- **マルチタスクが得意**: ゲーム進行、チップ管理、プレイヤー対応を同時に行う必要がある。
- **ストレスに強い**: 高速なゲーム進行、クレーム対応、長時間の立ち仕事に耐えられる。
- **人と接するのが好き**: プレイヤーとのコミュニケーションが仕事の重要な部分を占める。
- **ルールを守れる**: カジノの厳格なルールと手順に従って行動できる。

### 向いていない人

- 計算が極端に苦手（暗算に自信がない）
- 人前で作業することに強い緊張を感じる
- 不規則な勤務時間（夜間・週末・祝日勤務）が難しい
- 立ち仕事が身体的に困難

### ディーラーの魅力

- **チップ収入**: 基本給＋チップで高収入が可能。ラスベガスの繁盛店ではチップだけで年収$50,000超えも。
- **グローバルな職場**: 世界各国のカジノで働ける国際的な資格。
- **キャリアパス**: 経験を積めばピットボス→シフトマネージャー→カジノマネージャーへの昇進も。
- **人との出会い**: 世界中の様々な背景を持つ人々と交流できる。
