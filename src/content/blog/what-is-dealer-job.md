カジノディーラー。華やかなイメージの一方で、「実際の仕事内容は？」「自分に向いているか？」という疑問を持つ人は多い。僕は元公務員で現場経験ゼロだが、教材を作るために調べていくうちに「思っていたのと違う」ことが多かった。第三の視点から、ディーラーの実像を整理する。

### ディーラーの1日

| 時間 | 業務 |
|------|------|
| 出勤30分前 | 更衣、身だしなみチェック |
| 出勤 | 挨拶、テーブル担当確認 |
| セッション1 | 45〜60分のゲーム進行（休憩なし） |
| 休憩 | 20〜30分 |
| セッション2 | 再びテーブルに |
| 退勤 | チップ精算、報告書 |

これを8時間シフトで4〜5回繰り返す。立ちっぱなし、休憩は短い。役所のデスクワークより明らかにハードだ。

### 主な業務

- **ゲーム進行**: カード配布、ルーレットの球投げ、ダイス管理
- **チップ管理**: ベット受付、ペイアウト計算と支払い
- **不正防止**: カードのマーキング、チップ不正操作の監視
- **カスタマーサービス**: 挨拶、ルール説明、クレーム対応
- **テーブルメンテナンス**: カード交換、清掃、備品補充

### 向いている人

- 几帳面で正確（金銭を扱う。ミス厳禁）
- マルチタスクが得意（進行×管理×対応を同時に）
- ストレスに強い（クレーム、高速進行、立ち仕事）
- 人と接するのが好き（コミュニケーションが仕事の核）
- 夜型・不規則な生活に対応できる（24時間営業）
- チームワークを大切にできる（ピットボスや他ディーラーとの連携）

### やりがいと大変さ

やりがいは「エンターテインメントを提供する」という職人的な面。大変なのは体力負担と金銭管理のプレッシャー。華僑なイメージで入るとギャップが大きい。

## まとめ

ディーラーは「華やか」より「正確」が求められる仕事だ。向き不向きは明確に出るが、適性があれば他に代えがたいキャリアになる。僕のような公務員とは真逆の世界だが、構造として理解すると面白い。

**BAKUTOのそもそも論**
現場を知らない元公務員がカジノ業界の教材を作っている。
「普通の情報サイト」で終わらせたくない。構造を理解してほしい。

公務員時代も「1日の流れ」に沿って働いていたが、ディーラーのセッション制はそれ以上に厳格だ。役所の決済処理とディーラーのチップ管理、ミスが許されないという点では構造的に似ている。ただし、立ちっぱなし8時間の体力勝負は公務員にはない世界だ。
