## ブラックジャックのベーシックストラテジー完全解説

ベーシックストラテジーとは、ブラックジャックにおいて数学的に最適なプレイをまとめた戦略表のこと。1956年にBaldwin, Cantey, Maisel, McDermottが発表し、その後Julian BraunやEdward Thorpによって洗練された。

### 基本原則

ベーシックストラテジーの核心は、以下の4つの要素で決まる：

- 自分の手札の合計
- ディーラーのアップカード
- ソフト（Aを含む）かハード（Aなし）か
- スプリット可能なペアかどうか

### ハードハンドの戦略

| 自分の手札 | ディーラーのアップカード | アクション |
|-----------|----------------------|----------|
| 17以上 | any | Stand（絶対にヒットしない） |
| 13-16 | 2-6 | Stand（ディーラーのバーストに期待） |
| 13-16 | 7以上 | Hit（ディーラーが強いため勝負） |
| 12 | 4-6 | Stand（ディーラーのバースト確率が高い） |
| 12 | 2-3 | Hit |
| 11 | any | Double Down（最も強力なタイミング） |
| 10 | 2-9 | Double Down（ディーラーの弱い時は攻める） |
| 9 | 3-6 | Double Down |
| 8以下 | any | Hit（必ずヒット） |

### ソフトハンドの戦略

| 自分の手札 | ディーラーのアップカード | アクション |
|-----------|----------------------|----------|
| A-8以上（19以上） | any | Stand（十分に強い） |
| A-7（18） | 2-8 | Stand |
| A-7（18） | 9-A | Hit |
| A-6以下（17以下） | any | Hit or Double（状況による） |

### ペアのスプリット戦略

| ペア | アクション |
|------|----------|
| A-A | 常にスプリット（2枚のAは強力） |
| 10-10 | 絶対にスプリットしない（既に20） |
| 9-9 | 2-6, 8-9でスプリット |
| 8-8 | 常にスプリット（16は最弱、8はまし） |
| 7-7 | 2-7でスプリット |
| 6-6 | 2-6でスプリット |
| 5-5 | スプリットしない（代わりにDouble） |
| 4-4 | 5-6でスプリット（稀） |
| 2-2 / 3-3 | 2-7でスプリット |

### ディーラーが知っておくべきこと

ディーラーとしてベーシックストラテジーを理解していることは、以下の理由で重要：

- プレイヤーのミスを発見できる（ゲームプロテクション）
- プレイヤーの質問に的確に答えられる
- 自身のディーリングの流れを理解できる
- カードカウンティングの基礎となる
