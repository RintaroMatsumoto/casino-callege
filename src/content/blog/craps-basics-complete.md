## クラップスのルールとベット完全攻略

クラップスはカジノで最も複雑で、最も興奮するテーブルゲームだ。多くのプレイヤーが一度は挑戦するが、その複雑さゆえに敬遠されがちでもある。ディーラーとしては、このゲームを完璧にマスターすることで、他のディーラーとの差別化が図れる。

### ゲームの基本の流れ

1. **カムアウトロール**: シューター（ダイスを振る人）の最初のロール
2. **ナチュラル**: 7か11が出たらパスラインベットの勝ち
3. **クラップス**: 2, 3, 12が出たらパスラインベットの負け
4. **ポイント成立**: 4, 5, 6, 8, 9, 10が出たらその数字が「ポイント」になる
5. **ポイント再現 or セブンアウト**: ポイントと同じ数字が出れば勝ち、7が出れば負け

### 主要ベットのハウスエッジ

| ベット名 | ハウスエッジ | 説明 |
|---------|------------|------|
| パスライン | 1.41% | 基本ベット。最も人気 |
| ドントパス | 1.36% | パスラインの逆。やや有利 |
| オッズベット | **0%** | 唯一ハウスエッジがないベット |
| プレイス6/8 | 0.46% | 6か8に賭ける。低HEで人気 |
| プレイス5/9 | 1.11% | やや高いが許容範囲 |
| フィールド | 2.78%〜5.56% | 1ロールベット。配当設定により変動 |
| Any 7 | 16.67% | 高リスク。避けるべき |
| Any Craps | 11.11% | 2, 3, 12のどれかが出る |
| ハードウェイ | 2.78%〜9.09% | 同じ目での合計（Hard 4, 6, 8, 10） |
| プロップベット | 11.11%〜16.67% | センターベット全般。高HE |

### ディーラーとして覚えるべきこと

- **ペイアウト計算**: プレイスベット（7:6, 7:5, 9:5）、オッズベット（2:1, 3:2, 6:5）を瞬時に計算
- **スティックコール**: 出目の独特な宣告方法をマスター
- **チップ管理**: センターベット（プロップベット）の管理が最も複雑
- **ゲーム速度**: クラップスはカジノで最も速いゲームの一つ

### 初心者ディーラーへのアドバイス

クラップスは最初は圧倒されるが、以下の順序で学べば効率的だ：

1. まずパスラインとカムアウトの流れを完全に理解する
2. オッズベットの計算をマスターする（0% HEの重要性を理解）
3. プレイスベットの計算を覚える（最もよく使う）
4. 徐々にセンターベットを追加していく
5. スティックマンのコールを練習する

CasinoCallegeのPhase 3（クラップス）では、13のレッスンで段階的にクラップスを完全マスターできる。
