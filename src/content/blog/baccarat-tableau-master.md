## バカラのタブローを完全暗記する方法

バカラ（プントバンコ）において、ディーラーが最も習得に苦労するのが「タブロー」— 3枚目のカードを引くかどうかを決める固定ルール表である。このルールはプレイヤーの意思が一切介在しない、完全に機械的なものだ。

### タブローの基本構造

タブローは2つの部分から成る：
1. **プレイヤールール**: シンプル。最初の2枚の合計で決まる。
2. **バンカールール**: 複雑。プレイヤーの3枚目とバンカー自身の合計で決まる。

### プレイヤールール（覚えやすい）

| プレイヤーの最初の2枚 | アクション |
|---------------------|----------|
| 0-5 | 3枚目を引く |
| 6-7 | スタンド（引かない） |
| 8-9（ナチュラル） | ゲーム終了、3枚目は引かない |

プレイヤールールはこれだけ。一瞬で覚えられる。

### バンカールール（ここが勝負）

バンカーの3枚目ドローは、バンカー自身の合計と「プレイヤーの3枚目の値」の2つで決まる。

#### バンカーが3枚目を引く条件

| バンカーの合計 | プレイヤーの3枚目がこれなら引く |
|--------------|-----------------------------|
| 0-2 | **常に引く**（プレイヤーの3枚目に関係なく） |
| 3 | プレイヤーの3枚目が**1-2-3-4-5-6-7-9-10**のとき（※8以外は全部） |
| 4 | プレイヤーの3枚目が**2-3-4-5-6-7**のとき |
| 5 | プレイヤーの3枚目が**4-5-6-7**のとき |
| 6 | プレイヤーの3枚目が**6-7**のとき |
| 7 | スタンド（引かない） |
| 8-9（ナチュラル） | ゲーム終了 |

### 暗記のコツ

1. **「バンカー0-2は常に引く」**：これが基本。最もシンプル。
2. **「3は8以外全部引く」**：3は8だけが例外。
3. **「4は2-7」**：1と8-10は引かない。
4. **「5は4-7」**：範囲が狭まるイメージ。
5. **「6は6-7のみ」**：最も狭い。
6. **「7は絶対引かない」**：スタンド。

### 練習方法

1. **フラッシュカード**: バンカー合計＋プレイヤー3枚目の組み合わせをランダムに出題するアプリを使う。
2. **実戦想定**: 「バンカー4、プレイヤーの3枚目が6→引く」のように口に出しながら反復。
3. **間違いノート**: よく間違える組み合わせを記録し、重点的に復習。
4. **ペア練習**: 相手に出題してもらい、即答する練習。
