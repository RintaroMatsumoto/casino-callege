## §1 カジノの組織構造

`skill-tree`

### 1.1 カジノフロアのヒエラルキー

```
ゼネラルマネージャー (GM)
 └── カジノマネージャー
      └── シフトマネージャー（複数名、各8時間シフトを担当）
           └── ピットボス / フロアスーパーバイザー
                └── ディーラー
```

### 1.2 各役職の詳細

**ディーラー**（Dealer）: ゲームの進行役。テーブルに立ち、カード配布・ベット管理・ペイアウト計算を行う。給与は基本給＋チップ。北米では$25,000〜$80,000/年（ゲームと経験による）。

**ピットボス**（Pit Boss / Floor Supervisor）: 複数テーブルを監督する中間管理職。クレーム対応、高額ベットの承認、コンプ（無料サービスの提供判断）、新人ディーラーのトレーニングを行う。ディーラーから3〜5年で昇進可能。年収$40,000〜$70,000。

**シフトマネージャー**（Shift Manager）: 8時間シフト全体の運営責任者。ディーラーのシフト割り当て、トラブル対応、セキュリティとの連携。年収$60,000〜$100,000。

**カジノマネージャー**（Casino Manager）: カジノフロア全体の収益責任者。ゲームの選択、テーブル配置、スタッフ採用計画、年間予算の策定。年収$80,000〜$150,000+。

### 1.3 カジノの主要部門

| 部門 | 役割 | ディーラーとの関わり |
|------|------|-------------------|
| **サーベイランス** | 天井カメラによる24時間監視 | 常に監視されている。手信号はカメラに明確に |

| **ハウスキーピング** | 清掃、ユニフォーム管理 | 制服のクリーニング |

## §2 ディーラー面接・採用試験対策

### 2.1 採用プロセスの流れ

1. 書類審査（履歴書 + ゲーミングライセンス申請）
2. 背景調査（犯罪歴・与信情報）
3. 実技試験（チップカット、ディーリング実演）
4. 筆記試験（ルール・配当計算）
5. 面接（対人スキル・ストレス耐性）
6. 健康診断・薬物検査
7. 研修期間（通常4〜12週間）
8. 見習いデビュー（低額テーブルから）

### 2.2 実技試験の内容

| 試験 | 内容 | 合格基準 |
|------|------|---------|
| **チップカット** | 20枚のスタックを10-10に分割 | 5秒以内×3回連続成功 |
| **BJディーリング** | シューからカードを左→右に配布 | 20ハンド連続でミスなし |
| **ペイアウト計算** | BJ 3:2の口頭即答 | 10問中9問以上正解 |
| **手信号確認** | 各アクションの手信号 | 全6種正しく |
| **チップシャッフル** | 2スタックを1つに混ぜる | 10秒以内 |

### 2.3 面接でよく聞かれる質問と模範回答のポイント

**Q1: 「なぜディーラーになりたいのですか？」**
→ ゲームへの興味・対人サービスへの適性・キャリアとしての安定性を簡潔に。ギャンブル好きはNG。

**Q2: 「お客さんとトラブルになったらどうしますか？」**
→ 冷静に対応し、即座にピットボスを呼ぶ。自己判断での解決は避ける。

**Q3: 「ミスをしたらどうしますか？」**
→ 隠さずに即座に上司に報告する。誠実さと正確さが最も重視される。

**Q4: 「ギャンブル依存症についてどう思いますか？」**
→ 真剣な問題として認識し、兆候を見かけたら上司に報告する責任があると理解している。

## §3 ゲームプロテクション（完全版）

### 3.1 不正行為の類型と対策

ディーラーはゲームの公正さを守る最後の防衛線である。以下の不正を常に警戒する。

| 不正の種類 | 手口 | ディーラーの具体的な対策 |
|-----------|------|----------------------|
| **ベット後付け**（Past Posting） | 結果が判明した後にチップを追加する | 「No more bets」の宣言後はベットエリアのチップを凝視。手信号はアイ・イン・ザ・スカイへの証拠になる |
| **チップ抜き取り**（Chip Dragging） | 勝ったフリをしてチップを自分の山に加える | 支払い前に「どのベットが勝ったか」を指差し確認してからチップを動かす |
| **カードマーキング**（Card Marking） | 爪で傷をつける、インクで印をつける | カードを配る前に1枚ずつ表面を確認する習慣。定期的なデッキ交換 |
| **カード交換**（Card Switching） | 手持ちのカードとすり替える | デッキの色・シリアル番号の確認。プレイヤーがカードをテーブルの下に持ち込むことを禁じる |
| **エッジソーティング**（Edge Sorting） | カード裏面の印刷のわずかなズレを利用し、高価値カードの向きを識別する | カードをランダムに180度回転させて配る（2012年Phil Ivey事件以降の標準対策） |
| **コリュージョン**（Collusion / 共謀） | 複数のプレイヤーが協力してカジノを欺く | 不自然なプレイパターン（一方が常に負ける等）があれば報告 |

### 3.2 サーベイランスとの連携

カジノのサーベイランス（監視）部門は、天井のカメラでテーブルを24時間録画している。ディーラーは以下の点を常に意識する。

- 手信号はカメラに**はっきりと見える**ように行う
- チップの動きは**ゆっくりと、わざとらしく**見せる（スローモーションで確認できるように）
- 疑わしい事象があれば、**カメラに向かって**手信号でピットボスを呼ぶ
- 録画映像は紛争解決の最終証拠となる。自分の行動が映像でどう映るかを常に考える

## §5 責任あるギャンブリング

### 5.1 問題ギャンブルの兆候

ディーラーは以下の兆候を認識し、必要に応じてピットボスに報告する責任がある。

- 負けを取り戻そうと必死にベットを続ける（**チェイシング**）
- 予定時間を大幅に超えてプレイし続ける
- 感情的になり、通常とかけ離れた高額ベットをする
- 借金の話をする、または明らかに所持金以上の賭けをする
- 食事や睡眠をとらずに長時間プレイする

### 5.2 対応手順

1. 直接注意したり説教したり**しない**（ディーラーの権限外）
2. ピットボスに**静かに**報告する
3. ピットボスの指示に従う
4. カジノによっては、自主排除（**Self-Exclusion**）プログラムが用意されており、問題ギャンブルを自覚したプレイヤーが自ら「一定期間カジノに入れないようにする」申請ができる

### 5.3 カジノの法的責任

多くの地域では、カジノ運営ライセンスの条件として、責任あるギャンブリング対策が義務付けられている。問題ギャンブラーに賭けを続けさせることは、ライセンス剥奪のリスクにもつながりうる。ディーラーは、この仕組みの最前線のセンサー役である。

## §6 スロットマシン基礎（最低限の知識）

### 6.1 スロットマシンのビジネス規模

スロットマシンは、平均的な米国カジノの**収入の60〜70%**を占める最大の収益源である。テーブルゲームとは異なり、ディーラーが不要なため、人件費が極めて低い。このためカジノ経営において最重要の位置を占める。

### 6.2 基本用語

| 用語 | 意味 |
|------|------|
| **RTP**（Return to Player） | プレイヤーへの長期平均還元率。95% = $100賭けて平均$95戻る |
| **ボラティリティ** | 当たりの頻度と大きさ。高ボラ = 当たりは少ないが大きい |
| **ペイライン** | 当選判定に使うライン。ビデオスロットでは最大1,024本 |
| **ワイルド** | 他のシンボルを代用できる万能シンボル（ジョーカー的な役割） |
| **スキャッター** | ペイラインに関係なく出現するだけで有効なシンボル。ボーナスのトリガー |
| **プログレッシブジャックポット** | 複数台でリンクし、プレイのたびに賞金が積み上がる仕組み |
| **TITO** | Ticket-In Ticket-Out。紙チケットでスロットを操作する現金不要のシステム |

### 6.3 RTPの州別規制（米国）

| 州 | 最低RTP規制 |
|----|-----------|
| ネバダ | 75%以上 |
| ニュージャージー | 83%以上 |
| ミシシッピ | 80%以上 |

RTPが高いほどプレイヤーに有利だが、実際の設定値は各カジノが判断する。公表されないことが多い。

## §7 ディーラーの健康管理

### 7.1 身体的負担と対策

- **長時間の立ち仕事**: 8時間シフト中、30〜60分おきの休憩。クッション性の高い靴の着用
- **手首・指の疲労**: カード配布とチップ操作の繰り返し。シフト前後のストレッチが効果的
- **腰の負担**: 前かがみでのチップ回収。腰を落とす正しい姿勢の維持
- **目の疲れ**: カードの数字・チップの色・監視モニターの注視。定期的な遠くを見る休憩

### 7.2 メンタルヘルス

カジノは24時間営業の閉鎖空間で、人工照明・騒音・タバコの煙が充満している。加えて、負けて苛立つ顧客への対応や、ミスの許されない緊張感が続く。オフの日の完全な休息、日光を浴びること、趣味の時間の確保が長期的な健康維持に不可欠である。
