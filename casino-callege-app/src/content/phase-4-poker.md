# Phase 4: Poker — Texas Hold'em

> ポーカーは、他のカジノゲームと根本的に異なる。プレイヤーはカジノ（ハウス）と戦うのではなく、**他のプレイヤーと戦う**。
> ディーラーの役割は「対戦相手」ではなく「審判・進行役」である。
> この章では、テキサスホールデムのルール、ディーラー手順、ポット管理、レーキ計算までを網羅する。

**出典**: Wikipedia "Texas hold 'em", Wikipedia "Glossary of poker terms"

---

## §1 ゲームの概要

### 1.1 テキサスホールデムとは

各プレイヤーに2枚の**ホールカード**（※自分の手札。hole cards / pocket cards）が裏向きで配られ、さらに5枚の**コミュニティカード**（※全員が使える共有カード。community cards）がテーブル中央に表向きで順次公開される。プレイヤーは自分のホールカード2枚とコミュニティカード5枚の計7枚から、最も強い5枚の組み合わせを作る。

### 1.2 基本用語

- **ポット**（pot）: そのハンドで賭けられたチップの総額。勝者が全額を獲得する
- **ベッティングラウンド**（betting round）: 各カード公開後の賭けの機会。計4回ある
- **ショーダウン**（showdown）: 最終ラウンド後に残ったプレイヤーが手札を公開し勝者を決める段階
- **フォールド**（fold）: 手札を捨ててそのハンドから降りること。それまでに賭けたチップは戻らない
- **チェック**（check）: 賭けずに次のプレイヤーに回すこと。前に誰もベットしていない場合のみ可能
- **コール**（call）: 直前のプレイヤーと同額を賭けてハンドに残ること
- **レイズ**（raise）: 直前のベットより高い額を賭けること

### 2.1 スートの扱い

ポーカーにおいて、♠スペード、♥ハート、♦ダイヤ、♣クラブの4スートは**すべて同等**。スートによる強弱はない。フラッシュの比較では、含まれるカードのランク（数字の強さ）で勝敗を決する。

### 2.2 Aの特別な扱い

Aはストレートにおいて、**「A-K-Q-J-10」（最強のストレート）の先頭にも、「A-2-3-4-5」（最も弱いストレート、ホイール/バイスクル）の一部にも**なれる。ただし「Q-K-A-2-3」のようにAをまたいだストレートは成立しない。

## §3 ゲームの進行

### 3.1 ディーラーボタンとブラインド

ディーラーボタン（※Dealer Button。通常「D」と書かれた白い円盤）が、各ハンドの架空のディーラー位置を示す。実際のカード配布はカジノの専任ディーラーが行うが、ベッティングの順序はこのボタン位置を基準に決まる。ボタンはハンドごとに時計回りに1つずつ移動する。

`blinds-rotation`

`blind-growth`

ボタンの左隣のプレイヤーが**スモールブラインド（SB）**を、その左隣が**ビッグブラインド（BB）**を強制的に賭ける。SBは通常BBの半額。ブラインドは、何もしなければポットが0のままになるのを防ぎ、必ず取り合うチップが存在するようにするための仕組みである。

### 3.2 4つのベッティングラウンド

`betting-rounds`

**(1) プリフロップ（Pre-flop）**

`table-preflop`

全プレイヤーにホールカード2枚が裏向きで配られた後、最初のベッティングラウンド。BBの左隣（アンダー・ザ・ガン）から時計回りにアクション（※行動すること）が始まる。

**(2) フロップ（Flop）**

`table-flop`

デッキの一番上のカード1枚を**バーン**（※伏せて捨てること。不正防止の伝統的手法）した後、コミュニティカード3枚が表向きでテーブル中央に配られる。ボタンの左隣からベッティングが始まる。

**(3) ターン（Turn / Fourth Street）**

再度1枚をバーンした後、コミュニティカード1枚が追加される。SBの左隣からベッティング開始。

**(4) リバー（River / Fifth Street）**

再度1枚をバーンした後、最後のコミュニティカード1枚が追加される。同様にベッティング。

リバー終了時点で2人以上のプレイヤーが残っていれば、ショーダウンとなる。

### 3.3 ショーダウン（手札公開）

`table-showdown`

最終ベッティングラウンド終了後、残ったプレイヤーが手札を公開する。最後にレイズしたプレイヤーから先に公開するのが基本ルール。各プレイヤーは自分のホールカード2枚 + コミュニティカード5枚の計7枚から、最も強い5枚の組み合わせを選ぶ。ホールカードを1枚も使わず「ボード（コミュニティカード5枚）だけで勝負」することも可能である。

## §4 ディーリング手順

### 4.1 標準的な手順

1. シャッフル（最低2回のリフル + 1回のストリップ + カット）
2. カットカードをデッキの下部に挿入
3. 全プレイヤーにホールカード2枚を裏向きで配布（左から時計回りに1枚ずつ、2周）
4. プリフロップベッティング
5. **バーン1枚** → フロップ3枚を表向きで配布
6. フロップベッティング
7. **バーン1枚** → ターン1枚を表向きで配布
8. ターンベッティング
9. **バーン1枚** → リバー1枚を表向きで配布
10. リバーベッティング
11. ショーダウン → 勝者にポットを渡す

### 4.2 バーンカードの重要性

`burn-deal-timing`

バーンカード（各ラウンド前に1枚捨てるカード）には2つの目的がある。第一に、カードの裏面の傷やマークから次のカードを特定されるのを防ぐ。第二に、ディーリングのミスを発見しやすくする。バーンカードはプレイヤーに見せてはいけない。

### 4.3 よくあるディーリングミス

- **フロップを4枚配ってしまう**: 最も多いミス。3枚を一度にではなく、1枚ずつ数えながら配る習慣で防ぐ
- **バーンを忘れる**: ルーティンの不備。各ラウンド前に「バーン」と心の中で唱える
- **ボタンを飛ばす**: ハンド終了後にボタンを移動し忘れる。次のハンド開始時に必ずボタン位置を確認する

## §5 ポット管理とサイドポット

### 5.1 ポット計算の基本

ディーラーは各ベッティングラウンド終了時に、ポットの総額を把握しておく必要がある。

例: SB=$1, BB=$2。UTG（BBの左隣）が$6にレイズ。CO（カットオフ）がコール。BBがコール。

```
初期ポット: SB$1 + BB$2 = $3
UTGレイズ後: $3 + $6 = $9
COコール後: $9 + $6 = $15
BBコール後: $15 + $4 (BBは既に$2投入済みなので追加$4) = $19
レーキ5% ($3上限): min($19 × 5%) = min($0.95, $3) = $0.95
有効ポット: $18.05
```

### 5.2 サイドポット（オールイン発生時）

`sidepot`

プレイヤーが手持ちの全チップを賭けて**オールイン**した場合、残りのプレイヤーがさらにベットを続けると、**メインポット**（全員が争えるポット）と**サイドポット**（オールインプレイヤー以外が争うポット）の2つに分かれる。

例: A ($100オールイン), B ($200), C ($300)。Bが$200にレイズ、Cがコール。

```
メインポット: 3人 × $100 = $300
サイドポット: Bの追加$100 + Cの追加$100 = $200
Aが獲得できるのはメインポットのみ
```

## §6 レーキ（手数料）

`rake-flow`

### 6.1 レーキの種類

| 方式 | 計算方法 | 使用場面 |
|------|---------|---------|
| **ポットレーキ** | ポットの一定%（通常5%、上限$3〜5） | 通常のキャッシュゲーム |
| **タイムレーキ** | 30分ごとに固定額（$5〜8/人） | 高額テーブル |
| **トーナメントフィー** | 参加費の一部（10〜20%） | トーナメント |

### 6.2 レーキの徴収方法

ディーラーは、フロップ配布後（または初回ベッティング終了後）に、ポットからレーキ分のチップを抜き取り、ドロップボックスに投入する。これがカジノのポーカールーム収益となる。

## §7 確率の基礎

### 7.1 アウツと勝率

**アウツ**（outs）: 残りのデッキの中で、自分の手を改善できるカードの枚数。

**4倍・2倍の法則**（概算）:
- フロップ時の完成確率 ≒ アウツ × 4（%）
- ターン時の完成確率 ≒ アウツ × 2（%）

### 7.2 主なドローのアウツ数

`outs-flush`

| ドローの種類 | アウツ数 | フロップ→リバー完成率 |
|------------|---------|---------------------|
| フラッシュドロー（同スート4枚） | 9 | 約35% |
| オープンエンドストレートドロー（両端待ち） | 8 | 約31.5% |
| ガットショットストレートドロー（中抜け待ち） | 4 | 約16.5% |
| フラッシュ+オープンエンドの複合 | 15 | 約54% |

### 7.3 ポットオッズ

`pot-odds-calc`

ポットオッズ = コール額 / (コール後のポット総額)

この値より高い勝率があれば、コールは長期的にプラス期待値となる。

例: ポットに$100。相手が$50ベット。コールに必要な金額は$50。
ポットオッズ = $50 / $200 = 25%
→ 自分の勝率が25%以上あればコールすべき。
