# 日本IR（統合型リゾート）詳細

## 1. IRとは

**Integrated Resort（統合型リゾート）** は、カジノに加え、ホテル、国際会議場（MICE）、ショッピングモール、劇場、テーマパーク等を併設した大型複合リゾート。

- シンガポールの **Marina Bay Sands** と **Resorts World Sentosa** が成功モデル
- マカオのカジノ収入の約91%がバカラから（VIP中心）
- 日本のIR法はシンガポールモデルを参考に設計

出典: Wikipedia "Integrated resort"

---

## 2. 日本のIR法の経緯

| 年月 | 出来事 |
|------|--------|
| 2016年12月 | IR推進法（カジノ合法化法）成立 |
| 2018年7月 | IR実施法（IR整備法）成立。最大3地域まで認可 |
| 2020年1月 | 政府がIR基本方針を閣議決定 |
| 2021年 | 大阪・和歌山・長崎が区域認定申請 |
| 2023年4月 | 大阪IR（MGM Osaka）が国内初の区域認定を取得 |
| 2025年4月 | MGM Osaka メインリゾート建設開始（予定） |
| 2029-2030年 | **開業目標**（複数報道による） |

出典: Wikipedia "Casino hotel - Japan"

### 日本IRの基本ルール
- 入場料: 日本人・在留外国人は **6,000円/回**（外国人旅行者は無料）
- 入場回数制限: 週3回、月10回まで（日本人）
- カジノ面積: IR全体の3%以下または15,000m²以下
- 広告規制: 日本人向けのカジノ広告は厳しく制限
