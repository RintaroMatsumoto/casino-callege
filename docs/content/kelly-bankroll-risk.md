カジノゲームの数学的最適ベットサイズと資金管理。

## 1. Kelly基準 (Kelly Criterion)

### 1.1 定義

**ジョン・L・ケリーJr.** （1956年）がAT&Tのノイズ研究のために開発した公式。ギャンブルと投資における**最適ベットサイズ**を求める。

f* = (bp - q) / b

ここで:
- f* = バンクロールに対する最適ベット割合
- b = 得られるオッズ（「b to 1」、つまりベット1に対してbの利益）
- p = 勝率
- q = 負率 = 1 - p

### 1.2 フェアコインの例（単純化）

表が出たら2倍、裏が出たら0倍（b=1, p=0.5）:
```
f* = (1 × 0.5 - 0.5) / 1 = 0.0 → ベットすべきでない
```

表が出たら3倍、裏が出たら0倍（b=2, p=0.5）:
```
f* = (2 × 0.5 - 0.5) / 2 = 0.5 / 2 = 0.25
→ バンクロールの25%をベットすべき
```

### 1.3 BJでの応用（カードカウンティング）

推定アドバンテージが2%の場合:
```
f* = 0.02（単純化: アドバンテージ = f*）
→ バンクロールの2%をベット
```

実際には標準偏差を考慮した**Fractional Kelly**が使われる:
```
実践的ベット = f* × Kelly Fraction
Kelly Fraction = 0.25-0.50（1/4-1/2 Kellyが推奨）
```

**実例**: バンクロール$10,000、アドバンテージ+1%（TC=+2想定）:
```
Full Kelly: $10,000 × 0.01 = $100ベット
1/2 Kelly: $10,000 × 0.005 = $50ベット
1/4 Kelly: $10,000 × 0.0025 = $25ベット
```

### 1.4 ケリー基準の限界
1. **正確な確率が必要**: 推定誤差が破滅的な結果を招く
2. **対数効用関数を前提**: すべての人がこれに従うべきではない
3. **現実の制約**: 最小ベット・最大ベット・チップの最小単位
4. **時間選好**: 長期的な最適化は短期的な変動リスクを許容する

## 2. バンクロール管理の数学

### 2.1 リスクオブ・ルイン (RoR) の一般式

R = ((1 - EV/σ²) / (1 + EV/σ²))^(B/σ²)

**簡易近似**（EVが小さい場合）:
R ≈ e^(-2 × EV × B / σ²)

### 2.2 BJのリスクオブ・ルイン表

バンクロール（ベット単位）とリスクオブ・ルインの関係:

| バンクロール | RoR(HE=0.5%) | RoR(HE=1.0%) | RoR(Advantage=1%) |
|------------|-------------|-------------|------------------|
| 100 | 36.8% | 55.6% | 0.02% |
| 200 | 13.5% | 30.9% | <0.001% |
| 500 | 0.67% | 5.3% | — |
| 1,000 | 0.005% | 0.28% | — |
| 10,000 | — | — | — |

出典: Wizard of Odds Risk of Ruin appendix

### 2.3 クラップスのリスクオブ・ルイン

| バンクロール（ベット単位） | パスライン RoR | Any 7 RoR |
|------------|-------------|-----------|
| 100 | 24.3% | 81.1% |
| 200 | 5.9% | 65.8% |
| 500 | 0.04% | 35.1% |
| 1,000 | — | 12.3% |

**洞察**: Any 7のようにハウスエッジが高いベットは、同じバンクロールでも破産確率が劇的に高い。

## 3. 危険なベットシステムの数学的批判

### 3.1 マルチンゲール法

**ルール**: 負けるたびにベットを2倍にする。

**問題点の数学的分析**:
```
初回ベット: $10
連続損失確率: (20/38)^n（米国ルーレット赤黒）
5連敗の確率: (20/38)^5 = 4.0%
5連敗時の損失: 10+20+40+80+160 = $310
6回目のベット: $320
テーブルリミットやバンクロール制限で破綻
```

**期待値分析**:
```
勝つ確率（5回以内に1回勝つ）= 1 - 0.04 = 96%
その場合の利益 = $10（初回ベット額）
負ける確率 = 4%
その場合の損失 = $630

期待値 = 0.96 × $10 + 0.04 × (-$630)
       = $9.60 - $25.20
       = -$15.60（マルチンゲールはハウスエッジを変えない）
```

### 3.2 1-3-2-6システム

**ルール**: 勝つごとにベットを1→3→2→6単位に増やす。

| 結果 | 確率 | 損益 |
|------|------|------|
| 1回目負け | 50.7% | -1単位 |
| 1勝→2回目負け | 24.9% | +1単位 |
| 2勝→3回目負け | 12.5% | +2単位 |
| 3勝→4回目負け | 6.3% | 0単位 |
| 4連勝 | 5.6% | +12単位 |

```
期待値 = (-1 × 0.507) + (1 × 0.249) + (2 × 0.125) + (0 × 0.063) + (12 × 0.056)
       = -0.507 + 0.249 + 0.250 + 0 + 0.672
       = +0.664単位 → 4連勝の確率を過大評価していないか？

実際の確率（BJ想定、勝率42%）:
4連勝 = 0.42^4 = 3.1%
修正期待値 = (-1 × 0.58) + (1 × 0.42×0.58) + (2 × 0.42²×0.58) + (0 × 0.42³×0.58) + (12 × 0.42⁴)
           = -0.580 + 0.244 + 0.205 + 0 + 0.031
           = -0.100単位
```

**結論**: システムはハウスエッジを変えられない。負けるときもあれば勝つときもあるが、長期的にはシステムなしと同じ期待値。

## 4. バンクロール管理の実践ガイド

### 4.1 ゲーム別推奨バンクロール

| ゲーム | 希望RoR | 必要バンクロール（ベット倍数） |
|-------|---------|------------------------|
| BJ（基本戦略） | <5% | 200倍 |
| BJ（基本戦略） | <1% | 500倍 |
| BJ + カードカウンティング | <5% | 500倍 |
| ルーレット（欧州）偶数ベット | <5% | 100倍 |
| ルーレット（米国）偶数ベット | <5% | 200倍 |
| ルーレット（米国）ストレートアップ | <5% | 2,000倍 |
| クラップス パスライン（最大オッズ） | <5% | 100倍 |
| クラップス Any 7 | <5% | 実質不可能 |

### 4.2 損失制限と利益目標

**数学的な根拠**: 損失制限も利益目標も、長期的な期待値を変えることはできない。しかし、**心理的な破綻**を防ぐ効果はある。

**推奨ルール**:
- 1セッションの損失上限: バンクロールの10%
- 1回のセッション時間上限: 2-3時間
- 利益確定ライン: バンクロールの20%増加で一旦休憩

## 5. 第2章: 高度な統計概念

### 5.1 検出力分析（Power Analysis）

カードカウンターが検出される確率を評価する。

**例**: カジノの監視チームが、ベットサイズと勝率の相関を監視している。
```
H₀: ベットサイズと勝率に相関なし
H₁: ベットサイズと勝率に正の相関あり

効果量: r = 0.5（強い相関）
サンプルサイズ: n = 100
有意水準: α = 0.05

検出力 = 1 - β = 約99.9%（100ハンドでほぼ確実に検出）
```

### 5.2 シミュレーションによる確率検証

モンテカルロ法の基本的な考え方:
```
n回のシミュレーションを行い、結果を集計
推定値の標準誤差 = σ / √n
n=10,000の場合、標準誤差は真の値の約1%
```

## 6. 演習問題

### Q1: ケリー基準
あなたはアドバンテージ+2%のBJ状況を見つけた。バンクロールは$5,000。
1/4ケリーを使う場合のベット額は？

### Q2: リスクオブ・ルイン
$10ベット、HE=0.5%、バンクロール$2,000のBJプレイヤーのRoRは？
（簡易近似式を使用）

### Q3: マルチンゲール批判
米国ルーレットで赤に$5ベットからスタートするマルチンゲール。
7連敗する確率と、その時の損失総額は？

### Q4: 検定
あるルーレットホイールで、1,000スピン中「赤」が550回出た。
このホイールに偏りがあると言えるか？（有意水準5%で検定せよ）

---

### 解答

**A1**: f* = 0.02。1/4 Kelly = 0.02 × 0.25 = 0.005
ベット額 = $5,000 × 0.005 = **$25**

**A2**: B = 200単位、EV = 0.005、σ = $0.99
R ≈ exp(-2 × 0.005 × 200 / 0.99²)
  ≈ exp(-2.04)
  ≈ **13.0%**

**A3**: 米国ルーレット赤の勝率 = 18/38 = 47.37%
7連敗の確率 = (20/38)⁷ = 0.0106 = **1.06%**
損失総額 = 5+10+20+40+80+160+320 = **$635**
8回目のベット = $640（テーブルリミット超過の可能性）

**A4**: 期待値 = 1,000 × (18/38) = 473.7回
標準偏差 = √(1,000 × 18/38 × 20/38) = 15.8
z値 = (550 - 473.7) / 15.8 = 4.83
|z| > 1.96 → **帰無仮説を棄却。偏りがあると言える。**
（ただし、1000回程度では多重比較の問題が残る）
