# 競合吸収レポート

単なる表面コピーではなく、各競合の核となる価値を抽出し、CasinoCallegeの血肉にする。

---

## 吸収した競合の核

### 1. Wizard of Odds — 数学的権威

**核となる価値**: 「全ての主張に数学的根拠がある」という信頼性

**吸収方法**:
- 全コンテンツにハウスエッジ・確率・期待値の数値を明記
- 出典を明示（例: 出典: Wizard of Odds / 出典: Wikipedia）
- 計算式も掲載し、自分で検証できるようにする
- バリアントごとのハウスエッジ差を表で示す
- 全ベットの確率・配当・ハウスエッジを網羅したリファレンス表を作成

**該当ファイル**: `docs/content/mathematics-reference.md`

### 2. BlackjackInfo (Ken Smith) — 構造化教育

**核となる価値**: 「24レッスンの学校形式」という学習のしやすさ

**吸収方法**:
- 各Phaseを「Lesson」単位に分割（全Phaseで合計50-70レッスン）
- 各レッスンに「学習目標」「本文」「練習問題」「理解度チェック」を配置
- Lesson 0: イントロダクション（何を学ぶか）
- Lesson 最終: まとめと復習テスト
- 「初級→中級→上級」のレベル分け
- 進捗バーで可視化

**該当ファイル**: `docs/curriculum-framework.md`

### 3. BJ21.com (Stanford Wong) — コミュニティ + 課金

**核となる価値**: 「専門家と学習者が交流する場」と「持続可能な収益モデル」

**吸収方法**:
- Q&Aデータベースを構築（よくある質問＋専門家回答）
- 段階的課金モデル: Free / Premium / Pro / Enterprise
- 定期的な業界ニュース（日本IR関連に特化）
- 会員限定コンテンツ（詳細戦略、練習問題解答、ビデオ）

**該当ファイル**: `data/q-and-a.json`, `docs/subscription-model.md`

### 4. Gambling.com — ローカライゼーション + ゲーミフィケーション

**核となる価値**: 「多言語展開」と「ユーザーエンゲージメント」

**吸収方法**:
- 日本語→英語→中国語の展開計画
- 学習進捗によるバッジ・ポイント・レベル
- 期間限定チャレンジ（7日間マスターコース等）
- リーダーボード（任意参加）

**該当ファイル**: `docs/gamification-design.md`
