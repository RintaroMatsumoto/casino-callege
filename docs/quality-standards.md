# CasinoCallege 品質基準 — 有料ノート品質ガイドライン

すべてのコンテンツが「有料で販売できる品質」であるための基準。
以下の4軸で常に自己検証すること。

---

## 基準1: 有料ノート品質

| チェック項目 | 合格基準 |
|------------|---------|
| フォーマット | Markdown整形。見出し階層が正しい。箇条書き・表・コードブロックを適切に使用 |
| 誤字・脱字 | 日本語の誤字・脱字・表記ゆれがない。英単語のスペルミスがない |
| 情報の完全性 | 1つのトピック内で情報が完結している。「後で出てきます」を極力避ける |
| 出典明記 | 数値には必ず出典を記載。外部情報には引用元を示す |
| 著作権 | 他サイトの丸写しがない。自分の言葉で再構成している |

## 基準2: 用語解説の徹底

すべての資料で、専門用語が登場したら**その場で解説する**。

**悪い例**:
> ディーラーはプレイヤーにカードを配る。

**良い例**:
> ディーラー（※カジノでゲームを進行する従業員。クロッピエとも呼ばれる）はプレイヤー（※お客さん）にカードを配る。

**ルール**:
- 専門用語が初登場したら `（※...）` 形式でその場で解説
- 特に難しい用語は `（△△語で□□という意味）` と補足
- 同じ用語が再度登場しても、前の解説を思い出せるように簡易補足を入れる
- `data/glossary-detailed.json` の用語と連携させる

## 基準3: 多角的観点

各トピックを最低3つの視点から解説する：

| 視点 | アイコン | 問いかけること |
|------|---------|--------------|
| 👤 プレイヤー | `👤 プレイヤーはこう考える` | 「これはプレイヤーにとってどういう意味？」 |
| 🎩 ディーラー | `🎩 ディーラーはこう動く` | 「ディーラーとしてどう対応すべき？」 |
| 🏢 ビジネス | `🏢 カジノはこう見る` | 「カジノ経営にとってこのルールの意味は？」 |
| 📖 初心者 | `📖 はじめての人へ` | 「知識ゼロの人にもわかる説明になってる？」 |

## 基準4: 子どもでもわかる

| チェック | 内容 |
|---------|------|
| アナロジー | 日常的な例えを使う（「これはお店の値引きと同じ」など） |
| 段階的説明 | 最初に「全体像」→次に「詳細」→最後に「応用」 |
| カタカナ語の抑制 | 「ベーシックストラテジー」には必ず「基本戦略」の日本語を併記 |
| 図の代替 | 図がなくても理解できるように、テキストで詳細に描写 |

## 執筆時の統一ルール

| 項目 | ルール |
|------|--------|
| 日本語表記 | カタカナ語は初出時のみ英語＋日本語併記。2回目以降は日本語のみでOK |
| 例: 初出 | 「ベーシックストラテジー（Basic Strategy／基本戦略）」 |
| 例: 2回目 | 「基本戦略（ベーシックストラテジー）」 |
| 数字 | 3桁区切りはカンマ。小数点はピリオド |
| 強調 | `**太字**` は重要用語・結論。`*斜体*` は外来語・引用 |
| 用語リンク | 用語集に載っている用語は「※」で参照を示す |
