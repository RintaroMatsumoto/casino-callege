# CasinoCallege 改善計画 — 最強への道

8競合サイトの徹底分析に基づく、CasinoCallegeを最強のカジノ教育プラットフォームにするための実装計画。

---

## コア戦略：6つの差別化軸

### 1. ディーラー教育に特化（誰もやっていない）
### 2. 体系的カリキュラム（BJInfoの学校形式 × WoOの網羅性）
### 3. 日本語最優先（日本IR市場を狙い撃ち）
### 4. インタラクティブツール（WoOの計算機 + BJInfoのトレーナー）
### 5. コミュニティ + サブスクリプション（BJ21の収益モデル）
### 6. モダンなUI/UX（全競合が古いデザイン）

## 短期的なアクションアイテム

### 今すぐやること（Phase A）
1. **Phase 0-6をNext.jsページに変換**（各Phaseを独立したルートに）
2. **フラッシュカードモード** の実装（用語集をめくれるように）
3. **進捗管理**（LocalStorage）の実装
4. **ペイアウト計算ミニゲーム** の実装
5. **ダークモード** 対応（Tailwindで容易）

### やらないこと（リソース集中）
- Next.jsから別フレームワークへの移行はしない
- バックエンド/DBは初期段階では不要
- 動画コンテンツは後回し（テキスト + インタラクティブを優先）
